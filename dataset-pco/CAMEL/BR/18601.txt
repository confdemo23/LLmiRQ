Replace assertMockEndpointsSatisfied() in doc
The documentation for [migrating tests from junit4 to junit5|https://camel.apache.org/components/3.18.x/others/test-junit5.html#_migrating_camel_tests_from_junit_4_to_junit_5] provides an [example|https://camel.apache.org/components/3.18.x/others/test-junit5.html#_simple_unit_test_example] that uses the assertMockEndpointsSatisfied() method but in the class [org.apache.camel.test.junit5.CamelTestSupport|https://github.com/apache/camel/blob/main/components/camel-test/camel-test-junit5/src/main/java/org/apache/camel/test/junit5/CamelTestSupport.java] the method is not implemented, the documentation should be updated to indicate that we are supposed to use MockEndpoint.assertIsSatisfied instead.
