String arguments passed to DefaultScriptOperations.execute() appear quoted in script
DefaultScriptOperations.execute() calls convertScriptArgs() to convert the script&amp;apos;s arguments. For Strings it performs a local single-quoting.
This is necessary for DefaultScriptOperations.call() arguments (as the function call is constructed as a string) but not for execute() (as the arguments will be transfered in a DBObject.
