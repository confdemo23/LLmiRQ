Add serialVersionUID to ListRegionsOnServerFunction
The SDG ListRegionsOnServerFunction class used by the &lt;gfe-data:datasource/&gt; element in the SDG Data XML namespace implements the GemFire Function interface, which is indirectly java.io.Serializable by implementing GemFire&amp;apos;s Identifiable interface.  As a result, ListRegionsOnServerFunction should set the serialVersionUID property to avoid any ill de/serialization side-effects.
