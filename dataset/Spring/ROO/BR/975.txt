addon-web-mvc-jsp using incorrect package name
The web MVC JSP add-on currently uses a top-level package of org.springframework.roo.addon.mvc.jsp. It should be org.springframework.roo.addon.web.mvc.jsp in order to be consistent with the artifactId (which is correctly org.springframework.roo.addon.web.mvc.jsp) and the add-on directory name (which is correctly addon-web-mvc-jsp).
