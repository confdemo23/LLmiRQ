NPE in GwtMetadata for @RooEntity(versionField = "")
GwtMetadata.resolveEntityInformation(GwtMetadata.java:142) throws NPE trying to access entityMetadata.getVersionField() since there is no version field defined. By default DBRE annotates entities with @RooEntity(versionField = ""). At the same time EntityMetadata.getVersionField() return null in this case. This causes GwtMetadata to e caught with NPE.
