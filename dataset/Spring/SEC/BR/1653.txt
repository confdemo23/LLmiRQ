spring security sample war config < remember-me / > fails to properly register the requested bean
spring security config : &lt; remember-me / &gt; yields an error "Factory bean &amp;apos;org.springframework.security.userDetailsServiceFactory&amp;apos; not found".
this can be reproduced by import spring-security-samples-tutorial-3.1.0.M2.war from spring 3.1.0.M2 distribution on a freshly installed STS 2.5.1 installer from the spring website. After import -&gt; war, add spring nature and enable "Load NameSpaceHandlers and XSDs from projects classpath" from project properties.
