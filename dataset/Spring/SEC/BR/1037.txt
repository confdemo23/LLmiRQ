LdapAuthenticationProvider doesn&apos;t support messageSource updates/modifications
In order to allow localisation and personnal messages&amp;apos; definition org.springframework.security.providers.ldap.LdapAuthenticationProvider should implement org.springframework.context.MessageSourceAware.
At this time it&amp;apos;s not the case, so it is impossible to modify messages and use messages&amp;apos; localisation for this object (see 6.1 in spring security documentation http://static.springsource.org/spring-security/site/reference/html/supporting-infrastructure.html#localization).
By the way this object also doesn&amp;apos;t implement org.springframework.beans.factory.InitializingBean.
