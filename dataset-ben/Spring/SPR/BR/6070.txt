Differing types for getter/setter of CookieGenerator.cookieMaxAge causes configuring to fail.
CookieGenerator.cookieMaxAge's setter uses the primitive type 'int', but the getter uses 'Integer'.  This seems to stop this property from being configured through the spring configuration xml file.
If it's important, I'm actually using a subclass of CookieGenerator.
